# Brians API

To install:
```shell
$ yarn install
```
or
```shell
$ npm install
```

## ENV
Add a file `.env.local` to the root of the project with:

```shell
MONGO_URI=<your MongoDB connection string>
PORT=<any port you wish to use>
DEBUG=express:server
```

To run in dev mode:
```shell
$ yarn run dev
```
or
```shell
$ npm run dev
```
